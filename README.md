# Xrayception
 COVID Detection using Xception on Lung X-rays.

### Running
 To run the Flask App, after cloning the repository: 
```
git install lfs
git lfs pull
```
 to download the `xception_model.h5` file.\
 Run app.py to view the app locally at http://127.0.0.1:5000/
